# nexco_2rinwari_bulk_app

【ETC二輪車限定】2026二輪車定率割引の申し込みを「自動」で「一括」に行うアプリケーションです。

## 概要

NEXCO中日本が提供している以下の割引制度は、休日（土日・祝日）ごとに1日単位で
申し込みを行う必要があり、年度を通して利用しようとすると同じ操作を何十回も
繰り返さなければならず、UXが非常に悪い状態になっています。

* [【ETC二輪車限定】2026二輪車定率割引](https://hayatabi.c-nexco.co.jp/drive/detail.html?id=164&=1711517422953#FORM_LOGIN)

本アプリケーションは、早旅サイトへのログインから、申し込み可能な休日の洗い出し、
申請・承認までを自動でまとめて行います。

## 構成

```text
src/
  main.py              # エントリーポイント
  hayatabi/
    constants.py       # 共通定数（URL・年度依存IDなど）
    exceptions.py      # アプリ固有の例外
    models.py          # データモデル（AccountInfo）
    auth.py            # ログイン処理
    user_info.py       # ETCカード・車載器情報の取得
    holidays.py        # 申し込み対象となる休日の算出
    discount.py        # 2輪割の一括申請（申請〜承認）
    gui.py             # GUI（メインウィンドウ）
```

## セットアップ

実行に必要なパッケージは以下のコマンドでインストールできます。

```sh
sh pipinstall.sh
```

## 実行方法

```sh
python src/main.py
```

起動すると、早旅サイトのログイン情報（メールアドレス・パスワード）を入力する
ウィンドウが表示されます。ログインすると、申し込み可能な休日への一括申請が
自動的に開始されます。

## exe化（Windows用実行ファイルの作成）

[PyInstaller](https://pyinstaller.org/) を使用して、Windows用の実行ファイルを
作成できます。

```sh
pyinstaller HayatabiWindow.spec
```

`dist/` 配下に実行ファイルが生成されます。実行ファイルを配布・実行する際は、
アイコン表示に使用する `assets` フォルダを実行ファイルと同じ場所に配置してください。
